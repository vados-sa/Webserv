Como usar

Como usar este arquivo

Salve como site2.html (ou index.html) dentro do diretório raiz do segundo servidor. Exemplo de configuração (Nginx):

server {
    listen 8081;
    server_name segundo.local;
    root /var/www/site2;
    index index.html;
}

No seu webserv, aponte o server_name e a root para onde este arquivo estiver.
